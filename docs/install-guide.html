FlowGuard — Installation Guide

Complete setup: Git · VS Code · Node.js · Claude Code · FlowGuard

Windows 10 / 11 v2.0.2 by EHZ-AI
Before you start: Make sure you have a stable internet connection and an Anthropic account (console.anthropic.com). All software below is free.
Installation Steps — in order
1
Install Git
Version control · required by Claude Code ⬇ git-scm.com
Go to git-scm.com/download/win → download the installer → run it → keep all defaults → click Finish.
Verify in terminal: git --version
2
Install VS Code
Recommended code editor ⬇ System Installer (direct)
On code.visualstudio.com → click the dropdown arrow next to Download → choose Windows x64 System Installer.
⚠️ Do not use the "User Installer" — it will warn you when running as Administrator.
Run the System Installer → Next Next → check "Add to PATH" → Finish.
Verify: code --version
3
Install Node.js
JavaScript runtime · required for Claude Code & FlowGuard ⬇ nodejs.org
Go to nodejs.org → download the LTS version (recommended) → run installer → keep all defaults → Finish.
Verify: node --version and npm --version
4
Install Claude Code CLI
via npm 📄 Official Docs
Open Command Prompt or PowerShell and run: npm install -g @anthropic-ai/claude-code Wait 1–2 minutes for the installation to complete.
5
Login to Anthropic
One-time authentication 🔑 console.anthropic.com
Run in terminal: claude A browser window opens → sign in with your Anthropic account → authorize → return to terminal.
✓ Claude Code is ready. Try: claude "hello"
6
Install FlowGuard
Security monitor for Claude Code
Run FlowGuard-Setup.exe as Administrator → follow the setup wizard → FlowGuard registers itself automatically as a Claude Code hook.

After installation:
  • FlowGuard icon appears in System Tray 🟢
  • Open Dashboard → every Claude Code action is now monitored
  • Configure Telegram alerts in Settings for real-time notifications
